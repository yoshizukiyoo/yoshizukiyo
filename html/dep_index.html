타이포 .txt_*

  • 스타일 미지정시 .txt_md 동일 - 사이즈 16px, 두께 200 (Light), 행간 1.75, 자간 0
  • 영문, 숫자 : 스포카 한 산스 / 한글 : 노토 산스 / 기본 적용되어 있으므로 영문 및 숫자에 개별 스타일링할 필요 없음

본문 타이포

.txt_xxl 우체국 금융 차세대 채널 40px/Medium(500)/-0.05em

.txt_xl 우체국 금융 차세대 채널 26px/Medium(500)

.txt_lg 우체국 금융 차세대 채널 20px/Medium(500)

.txt_sm 우체국 금융 차세대 채널 14px/Light(200)
