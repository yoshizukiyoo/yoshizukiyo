02퀵메뉴 서비스 안내

퀵메뉴는 서비스 모든 페이지에서 확인 가능합니다. 퀵메뉴를 열거나 닫는 기능 표시
  • 투자성향분석 - 로그인 화면 투자자정보를 제공
  • 펀드이용가이드 - 사이트 이용안내 정보를 제공
  • 최근 본 펀드 - 카트에 담겨진 펀드상품정보를 제공하는 영역 표시(카트페이지로 이동하는 영역표시 비교하기, 장바구니 보기, 관심상품 보기)
  • 펀드검색 - 펀드검색 페이지로 이동
  • 추가매수 - 추가매수 페이지로 이동
  • 환매신청 - 환매신청 페이지로 이동
  • 용어사전 - 용어사전 새창열림
  • 투자권유준칙안내
  • 펀스 수시공시실
  • 고객센터 - 인터넷뱅킹 고객센터로 이동
  • 인증센터 - 인터넷뱅킹 인증센터로 이동
  • 챗봇(우정톡톡)상담 - 챗봇(우정톡톡)새창열림
