전체 FAQ

  • 암호화 진행 중 오류가 발생하였습니다. 라고 표시될 경우 해결방법은?
    프로그램 다운로드 시 다른 프로그램과 충돌 등으로 인해 프로그램이 손상될 수 있습니다. 이런 경우는 제어판>프로그램 추가/제거에서 ‘휴대폰인증서(저장)서비스’나 ‘UBIKey’ 프로그램을 삭제하시고 다시 서비스를 시작하시면 됩니다.
  • 마우스 커서가 입력되지 않거나, 인증서 암호 입력하면 화면이 무반응일 경우 해결방법은?
    • 인증서 발급기관과 유효기간을 확인해 보시기 바랍니다. 해당사이트에서 사용할 수 없는 인증서이거나 인증서 만료일이 지나 폐기된 인증서는 인증서 PC전송 후 인증서가 보이지 않을 수 있습니다. - 은행에서 사용 가능한 인증서: 은행/보험/신용카드용(개인), 범용(개인)
    • 해당 사이트의 키보드 보안 프로그램 (ex. AnySign등) 삭제 후 재설치 하여 이용하시기 바랍니다.
  • 이미 만료된 암호화 page입니다. 오류가 발생된 경우 해결방법은?
    서비스 중단 후 이용방법에 따라서 다시 절차를 수행하세요
  • 로그인 화면 접속 시 프로그램을 설치했는데도 계속 설치하라고 나오는 경우 해결방법?
    PC가 인증서 송수신 상태라면 SMS를 연결하지 않아도 이용 가능 합니다. PC에서 정보를 입력한 후, 휴대폰인증서 서비스 앱을 실행 한 후, [서비스 시작]을 선택하면 됩니다.
  • 공동인증서 발급(갱신)할 때 내부모듈 객체 초기화 실패하였습니다. 개체가 속성이나 메서드를 지원하지 않습니다. 키를 지정된 상태에서 사용하기에 부적합합니다. Windows Installer 서비스에 엑세스 할 수 없습니다. 라고 나오는 경우 해결방법은?
    인증서 암호 입력에 대한 제한은 없습니다.

우체국 보험 > 세제

  • 우체국의 보장성보험에 가입하고 있습니다. 소득,세액공제 한도는 얼마인가요?
    근로소득자에 한해 당해 연도에 납입한 보험료에 대해 연말정산시 100만원을 한도로 12% 세액공제 가능합니다. (사업소득자 및 일용근로자 제외) 
    ※ 소득, 세액공제 한도는 세법 개정 시 변경 될 수 있음
  • 연금저축보험의 연간 소득공제한도도 100만원 인가요?
    연금저축의 소득공제 한도는 당해 연도에 납입한 보험료 합계액을 연 400만원 한도 내에서 12% 세액공제 가능합니다. 
    ※ 종합소득금액 4,000만원 이하, 총급여액 5,500만원 이하는 15%로 세액공제
    ※ 연금저축만 납입했을 경우에는 한도 400만원, 퇴직연금에 납입한 금액을 합쳐서는 700만원을 한도로 공제가능
    ※ 소득,세액공제 한도는 세법 개정 시 변경 될 수 있음
  • 개인연금저축의 연금 수령 시 세제 적용은 어떻게 되나요?
