타이포 .txt_*

  • 스타일 미지정시 .txt_md 동일 - 사이즈 16px, 두께 200 (Light), 행간 1.75, 자간 0
  • 영문, 숫자 : 스포카 한 산스 / 한글 : 노토 산스 / 기본 적용되어 있으므로 영문 및 숫자에 개별 스타일링할 필요 없음

본문 타이포

상하단 간격 미포함되어있으므로 간격 설정은 멀티클래스 조합 사용

.txt_xxl 우체국 금융 차세대 채널 40px/Medium(500)/-0.05em

.txt_xl 우체국 금융 차세대 채널 26px/Medium(500)

.txt_lg 우체국 금융 차세대 채널 20px/Medium(500)

.txt_md 우체국 금융 차세대 채널 16px/Light(200) (스타일 미지정시 기본)

.txt_sm 우체국 금융 차세대 채널 14px/Light(200)

한 문장 내에 폰트 사이즈 혼용 사용 예시 

							<p class="txt_xxl"></p>
							<p class="txt_xl"></p>
							<strong class="txt_lg"></strong>
							<span class="txt_md"></span>
							<span class="txt_sm"></span>

  • 콘텐츠 영역은 기본적으로 닫혀있으며 버튼 클릭시 노출됨
	<!-- 버튼에 data-toggle-btn 애트리뷰트 추가 후 값 지정 -->
	<button type="button" class="btn btn_default btn_xs ml_2" data-toggle-btn="sample">더보기 +</button>

	<!-- 버튼으로 제어할 콘텐츠를 감싸는 컨테이너에 data-toggle-content 애트리뷰트 추가 및 값 매칭 -->
	<div data-toggle-content="sample">
		...
	</div>
